# Stint — acceptance evidence (verbatim CLI transcript)

Generated by `npm run evidence`. Every block below is the exact stdout/stderr and
exit code of a `tt` command run against a fresh database with the clock pinned to
`2026-06-24T17:30:00Z` (so durations are reproducible). Sections map to the PRD §17
acceptance criteria.

## R1 / R2 — one open entry; surfaces never disagree

A timer started through `tt` is the single open entry; starting another atomically closes the first.

```console
$ tt start "auth refactor" --client "Client A" --project API --tag deep --at 2026-06-24T09:00:00Z
▸ running 08:30:00 · "auth refactor" · Client A / API
# exit 0
```

```console
$ tt status
▸ running 08:30:00 · "auth refactor" · Client A / API
# exit 0
```

Derived elapsed = now − start, computed on demand (R1, R3).

```console
$ tt start "code review" --client "Client A" --at 2026-06-24T16:00:00Z
▸ running 01:30:00 · "code review" · Client A
# exit 0
```

```console
$ tt list --today --all
ID  START                 END                   DUR       CLIENT/PROJECT  DESCRIPTION    BILL  FLAGS
1   2026-06-24T09:00:00Z  2026-06-24T16:00:00Z  07:00:00  Client A / API  auth refactor  yes
2   2026-06-24T16:00:00Z  —                     01:30:00  Client A        code review    yes
# exit 0
```

Exactly one entry is open (R2); the first was closed at 16:00.

## R5 — sleep yields a flagged entry, second-accurate spans, reversible subtract

_A 6m30s suspend→resume cycle was recorded on entry 2 (as `powerMonitor` would on a real machine)._

```console
$ tt sleep ls
ID  DESCRIPTION  SPANS  EXCLUDED  SOURCES
2   code review  1      00:00:00  event
# exit 0
```

The entry is flagged slept-through with a second-accurate span.

```console
$ tt sleep subtract 2
excluded 00:06:30 of slept time from entry 2
# exit 0
```

Subtract moves exactly the slept seconds into excluded; billable drops, raw is unchanged (R5).

```console
$ tt sleep subtract 2
restored entry 2: slept time no longer excluded
# exit 0
```

Reversible — running it again restores the prior excluded_seconds.
